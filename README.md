# Biz
This repository contains a questions and answers application

## Built With
* Ruby on Rails
* Devise Token Auth
* SQLite3
* Rspec
* Shoulda matchers
* Factory bot

## Features
* Create question
* View all questions
* Submit an answer to a question
* View question and available answers

## Setup
### Backend
* Clone repo
* `cd <repository-path>`
* `bundle install`
* `rails db:create`
* `rails db:migrate`
* `rails db:seed`
* `rails s -p 3001`
